Pre-reqs:
Microsoft .NET Framework 4.7.1 Dev Pack:
https://www.microsoft.com/en-us/download/confirmation.aspx?id=56119

Microsoft Visual Studio 2017 (Community is free):
https://www.visualstudio.com/downloads/

Steps:
Clone/Pull latest repo
Open .sln file in Visual Studio 2017


Required Packages:
Open Tools > Package Manager Console
--
Install MahApps.Metro Package Enter-
PM> Install-Package MahApps.Metro
--
Install Prism.Core Package-
PM> Install-Package Prism.Core -Version 7.0.0.396
--
Install ControlzEx Package-
PM> Install-Package ControlzEx -Version 3.0.2.4
--
Install Mysql.Data Package-
PM> Install-Package MySql.Data
--
Extended WPF Toolkit (datetime picker, etc)
PM> Extended.Wpf.Toolkit -Version 3.3.0
